JavaScript Cheat Sheet

This page shows the most useful JavaScript tags in beginner-friendly terms.

let / const / var

Declare variables.

let x = 5;
const y = 10;
var z = 15;
function

Defines a reusable block of code.

function greet() {
  alert("Hello!");
}
if / else

Conditional logic.

if (x > 10) {
  console.log("Big");
} else {
  console.log("Small");
}
for / while

Loops for repetition.

for (let i = 0; i < 5; i++) {
  console.log(i);
}
getElementById

Selects a DOM element.

document.getElementById("demo").textContent = "Hi";
addEventListener

Adds interaction handler.

button.addEventListener("click", greet);
innerHTML

Sets or gets HTML content.

element.innerHTML = "Hello";
alert / console.log

Display or log messages.

alert("Hi");
console.log("Hello");
Math.random / Math.floor

Generate random numbers.

let num = Math.floor(Math.random() * 10);
Array.push / Array.pop

Add/remove elements from array.

arr.push("item");
arr.pop();
map / filter

Transform and filter arrays.

arr.map(x => x * 2);
arr.filter(x => x > 5);
JSON.stringify / JSON.parse

Convert between objects and JSON.

let json = JSON.stringify(obj);
let obj = JSON.parse(json);
setTimeout / setInterval

Delays or repeats functions.

setTimeout(greet, 1000);
setInterval(greet, 2000);
try / catch

Handle runtime errors.

try {
  callThatMightFail();
} catch (e) {
  console.log(e);
}
fetch

Request data from APIs.

fetch("https://api.example.com")
  .then(res => res.json())
  .then(data => console.log(data));
