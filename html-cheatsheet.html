HTML Cheat Sheet

This page shows the most useful HTML tags in beginner-friendly terms.

<!DOCTYPE>

Tells the browser this page uses HTML5.

<!DOCTYPE html>
<html>

Wraps the entire webpage content.

<html lang="en">
  ...your page content...
</html>
<head>

Contains settings like the title and CSS links.

<head>
  <title>My Website</title>
</head>
<meta>

Defines information like character encoding.

<meta charset="UTF-8">
<title>

Sets the page title (shown in the browser tab).

<title>Welcome Page</title>
<body>

This is where all the content goes.

<body>
  Hello, world!
</body>
<h1> to <h6>

Heading levels — h1 is largest, h6 is smallest.

<h1>Main Heading</h1>
<h2>Sub Heading</h2>
<p> – Paragraph

Creates a block of text.

<p>This is a paragraph.</p>
<a> – Link

Creates a clickable hyperlink.

<a href="https://example.com">Visit Site</a>
<img> – Image

Displays an image.

<img src="image.jpg" alt="My photo" width="300">
<br> – Line Break

Skips to a new line in text.

First line<br>Second line
<hr> – Horizontal Line

Adds a horizontal line across the page.

<hr>
<strong> and <em>

Makes text bold or italic.

<strong>Important!</strong>
<em>Emphasized text</em>
<ul>, <ol>, <li> – Lists

ul = bullet list, ol = numbered list, li = each item.

<ul>
  <li>Apples</li>
  <li>Bananas</li>
</ul>
<table>, <tr>, <td>, <th>

Creates a table of information.

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alex</td>
    <td>22</td>
  </tr>
</table>
