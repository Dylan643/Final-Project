CSS Cheat Sheet

This page shows the most useful CSS tags in beginner-friendly terms.

color

Sets the text color.

color: red;
background-color

Sets the background color.

background-color: lightblue;
font-family

Defines the font style.

font-family: 'Arial', sans-serif;
font-size

Sets the size of the text.

font-size: 16px;
margin

Sets the outer space of an element.

margin: 20px;
padding

Sets the inner space of an element.

padding: 10px;
border

Defines border around elements.

border: 1px solid black;
display

Specifies layout (block, inline, flex, etc.).

display: flex;
flex

Specifies the flex properties of a flex item.

flex: 1;
position

Specifies element position method.

position: relative;
z-index

Sets stack order of elements.

z-index: 10;
animation

Defines animation effects.

animation: fadeIn 1s ease-in;
transition

Specifies transition effects.

transition: all 0.3s ease;
text-align

Aligns the text horizontally.

text-align: center;
box-shadow

Adds a shadow to the element box.

box-shadow: 0 4px 6px rgba(0,0,0,0.1);
